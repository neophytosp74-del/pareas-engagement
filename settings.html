Trello REST API credentials

Χρειάζονται μόνο για την επισύναψη του engagement letter στην καρτέλα ως αρχείο και για την ενημέρωση της περιγραφής. Αποθηκεύονται κρυπτογραφημένα στο Trello ανά χρήστη — δεν τα βλέπει κανείς άλλος.

Πώς παίρνω τα στοιχεία:
1. Πήγαινε στο trello.com/app-key και αντέγραψε το API Key.
2. Στην ίδια σελίδα, κάνε κλικ στο link "Token" για να εκδώσεις έναν προσωπικό token και επικόλλησέ τον εδώ.
